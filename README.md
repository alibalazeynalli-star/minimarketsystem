## İstifadə Qaydası:
- Giriş üçün: `username: student1`, `password: 1234`
- 3 dəfə səhv şifrə yazsanız, sistem 10 saniyə sizi gözlədəcək.
- Bütün əməliyyatlar (alış-veriş, login) `data/` qovluğunda log şəklində saxlanılır.
